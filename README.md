# SMC_NMA
Network meta-analysis (NMA) of seasonal malaria chemoprevention



# code
Contains R scripts for NMA for all outcomes and scripts used to generate forest plots + treatment ranking heat plots



# data
Contains data for all uncomplicated malaria and severe malaria. 'meta regression' folder contains data on covariates used for meta-regression. 'tolerability' folder contains data for all tolerabiltiy outcomes
